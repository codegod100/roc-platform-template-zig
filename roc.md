=== Doc Module Demo ===

## Welcome to the Doc Module

This demonstrates a **recursive document type** for building structured Markdown.

## Text Formatting

- **Bold** text
- *Italic* text
- Inline `code`
- [Links to websites](https://roc-lang.org)

## Code Example

```roc
main! = |_args| {
    Stdout.line!("Hello, World!")
    Ok({})
}
```
